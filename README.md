# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/pedron3to/easy-live-challenge.git
```

```bash
$ cd ..
$ cd easy-live-challenge
# Install Dependencies
$ yarn
```

```bash
$ yarn start
```

Go to http://localhost:3000 to see the result.
